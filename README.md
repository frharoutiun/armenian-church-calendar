# Armenian Church Calendar Calculator

This is a web-native JavaScript version of the Armenian Church Calendar Calculator, adapted from the original Java Applet / Swing source.

## Files

- `index.html` — the webpage structure
- `styles.css` — the visual layout and styling
- `app.js` — the calendar calculation logic and interface behavior
- `ArmenianChurchCalendarApplet.java` — the original Java source kept for reference
- `.nojekyll` — tells GitHub Pages to serve the files directly

## How to use locally

Open `index.html` in a browser.

## How to publish on GitHub Pages

Upload all files in this folder to a GitHub repository, then enable GitHub Pages from the repository settings.
